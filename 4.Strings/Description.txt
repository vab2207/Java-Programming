This folder has programs on Strings.

String_Basics:
1. StringCreation.java - Make a string
2. StringConcatenation.java - Join strings
3. StringEqualsVsDoubleEquals.java - equals() vs ==

String_Methods:
1. CharAtExample.java - Get letter by index
2. SubstringExample.java - Cut part of string
3. ReplaceExample.java - Replace word
4. SplitExample.java - Break sentence

StringBuffer:
1. StringBufferExample.java - Change string using buffer
2. InsertDeleteBuffer.java - Insert or delete letter

StringBuilder:
1. StringBuilderExample.java - Same as buffer but faster
2. ReverseUsingBuilder.java - Reverse string

Practice:
-PalindromeCheck.java - Word same front and back
-WordCount.java - Count words
-CountVowels.java - Count a, e, i, o, u
-ReverseWords.java - Reverse each word
